## كيفية المساهمة

1. انسخ المستودع (fork)
2. أنشئ فرعاً جديداً لكل ميزة أو تصحيح
3. أضف تحسيناتك أو توثيقك
4. أرسل طلب سحب (Pull Request)
5. سيتم مراجعة التغييرات ودمجها بعد الموافقة
